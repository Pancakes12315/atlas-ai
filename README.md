# Atlas AI

Atlas AI is an AI-native enterprise knowledge platform that helps organizations search, understand, and proactively discover knowledge across all enterprise systems.

## Features

- Enterprise Search
- AI-powered Answers
- Explainable Recommendations
- Knowledge Graph
- Proactive Intelligence

## Tech Stack

Backend
- Java
- Spring Boot

Frontend
- React
- TypeScript
- Tailwind CSS

Search
- Elasticsearch

AI
- RAG
- Vector Search

Database
- PostgreSQL

Infrastructure
- Docker
